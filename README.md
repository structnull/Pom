## Getting Started

### Prerequisites

- Rust (1.5x or later)
- Cargo

### Installation

Clone the repository:

```
bash
git clone https://github.com/yourusername/rust-pomodoro-app.git
cd rust-pomodoro-app
```


Build and run the application:

```bash
cargo build --release
cargo run --release
```
